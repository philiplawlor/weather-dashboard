About Weather Dashboard

Project Overview

This is a demo project showcasing how to create a weather dashboard using Flask and the OpenWeatherMap API. The application provides real-time weather information for locations worldwide.

Features

  • Get current weather data from the OpenWeatherMap API
  • Search for weather by city name or ZIP code
  • View detailed weather information including temperature, humidity, and conditions
  • Responsive design that works on desktop and mobile devices
  • View recent search history
  • Temperature display in Fahrenheit

Tech Stack

Backend
  • Python 3.8+
  • Flask web framework
  • SQLite database
  • SQLAlchemy ORM
Frontend
  • HTML5
  • CSS3 with Bootstrap 5
  • JavaScript (ES6+)
  • AJAX for API calls
APIs Used
  • OpenWeatherMap API for weather data
  • OpenWeatherMap Icons for weather conditions

Version

Current Version: 0.1.0

Release Date: June 5, 2025
